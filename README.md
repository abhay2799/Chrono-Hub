# ⏰ ChronoHub

**ChronoHub** is a modern multi-feature time management web application that combines **Clock, Alarm, Stopwatch, and Countdown Timer** into a single elegant interface.

The project is built using **Vanilla JavaScript with a modular architecture**, focusing on performance, clean code structure, and a premium UI experience.

---

## 🚀 Features

### 🕒 Clock

* Real-time analog clock
* Digital clock toggle
* Smooth second hand animation

### ⏰ Alarm System

* Create multiple alarms
* Toggle alarms on/off
* Edit alarms
* Delete alarms
* Snooze system
* Custom alarm sound upload
* Default alarm sound fallback
* Alarm popup interface

### ⏱ Stopwatch

* Millisecond precision timing
* Lap recording system
* Smooth time updates
* Start / Stop / Reset controls

### ⌛ Countdown Timer

* Custom countdown duration
* Live countdown updates
* Completion alert

### ⚙️ Settings Panel

* Dark Mode toggle
* Digital Clock toggle
* Auto Mode option
* Clean settings interface

---

## 🧠 Tech Stack

* **HTML5**
* **CSS3**
* **Vanilla JavaScript (ES6 Modules)**

No frameworks or libraries were used — everything is built from scratch.
